How Grid Layout Works in CSS3

The CSS Grid Layout Module offers a grid-based layout system, with rows and columns, making it easier to design web pages without having to use floats and positioning. A grid layout is created by declaring display: grid or display: inline-grid on a grid container element. As soon as we do this, all direct children of that element become grid items. All components in the layout are given equal size.

Rows and columns are defined by grid-template-columns and grid-template-rows properties. The grid-template-columns property specifies the number (and the widths) of columns in a grid layout. The values are a space separated list, where each value specifies the size of the respective column. View the next examples:

The property value auto distributes width among columns evenly in a row. The following css code snippet will result in a grid with 4 coloumns equal in width
grid-template-columns: auto auto auto auto;

we can offset the width value of the columns by explicitly defining widths for the columns
grid-template-columns: auto 10% auto auto;
This will generate a 3 by 4 grid where the second column occupies 10% of the entire available space and columns 1, 3, and 4 occupy equal amount of the remaining width.

Grid also introduces an additional length unit to help us create flexible grid tracks. The new fr unit that represents a fraction of the available space in the grid container.
